# WpfTaskTracker

WpfTaskTracker is a simple WPF desktop application used as a template
for software documentation exercises.

The goal of this project is not application development, but learning
how to extract technical documentation directly from source code.

## Technologies
- WPF (.NET 8)
- MVVM architectural pattern
- JSON file persistence
- File-based logging
